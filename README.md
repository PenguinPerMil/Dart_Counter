# Dart_Counter

Graphic interface to count score in dart games.

# Required 

python3 

tkinter  

** Windows **
``` python3 pip install tkinter ```

** Ubuntu **

``` sudo apt install python3-tkinter ```

  # Launch 

Go in /Dart_counter root

  ``` python3 ./main.py ```
